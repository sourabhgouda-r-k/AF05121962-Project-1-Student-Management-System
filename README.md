# Student Management System

A Java JDBC based console application connected with MySQL database to manage student records.

## Features

- Add Student
- View All Students
- Update Student Details
- Delete Student
- MySQL Database Connectivity using JDBC

## Technologies Used

- Java
- JDBC
- MySQL
- Eclipse IDE

## Database

Database Name: studentdb

Table Name: students

## Project Structure

src/sms/
- DBConnection.java
- Student.java
- StudentDAO.java
- Main.java

## How to Run

1. Create database in MySQL
2. Update DB password in DBConnection.java
3. Run Main.java
